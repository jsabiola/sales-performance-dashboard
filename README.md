# Sales Performance Dashboard (Power BI)

## Project Overview
This project analyzes retail sales data to uncover key business insights using Power BI.

## Dataset
- Source: UCI Machine Learning Repository
- Contains transactions, customers, products, and pricing information

## Data Cleaning
- Removed cancelled transactions
- Filtered zero-priced items
- Handled missing values
- Created revenue and time-based features

## Dashboard Features
- Total Revenue, Total Orders, Avg Order Value
- Monthly Sales Trend
- Sales by Country
- Top Products Analysis
- Interactive slicers (Year, Country)

## Key Insights
- Revenue is concentrated in a few countries
- Top products contribute most of the revenue (Pareto effect)
- Sales show seasonal patterns
- High-value customers drive significant revenue

## Tools Used
- Power BI
- DAX
- Excel
